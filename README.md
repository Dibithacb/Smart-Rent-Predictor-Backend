Smart rent predictor - Backend

Tech stack
----------
Node.js with Express.js
MongoDB with Mongoose ODM
JWT for authentication
bcryptjs for password hashing
cookie-parser for cookie handling
cors for cors-origin requests

Front end hosting link:-
https://smart-rent-predictor-6u5u.vercel.app/


Backend hosting link:-
https://smart-rent-predictor-backend.onrender.com

Security best practices:-
Passwords hashed with bcrypt (10 rounds)
JWT tokens with 3-day expiration
Input validation of all end points
Password reset tokens expire after 1 hour
Random token generation with crypto
